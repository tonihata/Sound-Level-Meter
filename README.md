# Sound-Level-Meter

Sound level meter made with own pcb. I used kicad to make the pcb and to manufacture it i used milling. For the software i used Arduino IDE. The case was made using the Autodesk Inventor.

I used the ATMEGA328P -microchip and the MAX4466 -microphone.
